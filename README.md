# Terminal Nilam - Network Monitoring System

> **Sistem Monitoring Perangkat Jaringan & CCTV**

## 📋 Tentang Aplikasi

Aplikasi mobile untuk monitoring real-time perangkat jaringan dan kamera CCTV. Dibangun dengan Flutter untuk memberikan tampilan yang responsif dan mudah digunakan.

## ✨ Fitur

### 📡 Monitoring
- Pemantauan status CCTV secara real-time
- Monitoring perangkat MMT (Massive MIMO Tower)
- Pemantauan perangkat jaringan
- Pengecekan konektivitas perangkat
- Update status otomatis

### 📊 Manajemen
- Dashboard dengan statistik perangkat
- Manajemen data perangkat
- Notifikasi status perangkat
- Riwayat alert dan perubahan status

### 👤 User Management
- Sistem autentikasi pengguna
- Manajemen profil dan akun
- Keamanan akses berlapis

## 🛠️ Teknologi

- **Flutter** - Cross-platform mobile framework
- **Dart** - Programming language
- **REST API** - Backend communication

## � Cara Menggunakan

1. Clone repository ini
2. Jalankan `flutter pub get` untuk install dependencies
3. Jalankan aplikasi dengan `flutter run`

## 📱 Antarmuka Aplikasi

Aplikasi menyediakan interface yang user-friendly untuk:
- Dashboard dengan overview semua perangkat
- Monitoring detail status setiap perangkat
- Notifikasi real-time
- Manajemen data perangkat

## 🎯 Use Case

Aplikasi ini cocok untuk:
- Network Administrator yang perlu monitoring perangkat
- Security Team untuk pemantauan CCTV
- Facility Management untuk monitoring infrastruktur
- IT Operations untuk tracking availability perangkat
